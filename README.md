## 📜 Presentació del Projecte
**Archivum Vestigia** és una experiència web interactiva que combina la divulgació científica amb la gamificació. Dissenyada com una "càpsula del temps", la plataforma permet explorar fitxes tècniques d'espècies extintes i posar a prova els coneixements amb un qüestionari d'alta dificultat.

> [!IMPORTANT]
> Aquest projecte utilitza **JavaScript Vanilla** per a la lògica de correcció i el control dinàmic d'imatges.

---

## 🛠️ Detalls Tècnics

### 📁 Estructura de l'Arxiu
- `index.html`: Hub principal i interfície de l'examen.
- `script.js`: Lògica de l'algoritme de puntuació i manipulació del DOM.
- `CSS/style.css`: Full d'estils amb disseny *Dark Mode* juràssic.
- `Imagenes/`: Magatzem de recursos gràfics i GIFs de seguretat.
- `*.html`: Mòduls individuals d'aprenentatge (Mosasaurus, Spinosaurus, etc.).
